# Portfolio Design Spec: The Architect's Vault

## 1. Visual Theme: "Premium Technical Minimal"
*   **Palette**: Keep the dark mode base (`#0d1117`) but add **"Electric Cyber"** accents (`#ff00ff` or `#00f2ff`) for interactive elements.
*   **Typography**: Use high-readability monospace fonts for technical data (e.g., `Fira Code`, `JetBrains Mono`) and bold, assertive sans-serif for headings (e.g., `Inter`, `Manrope`).

## 2. The Hero Section (Refined)
*   **Old**: "Building Robust mobile applications."
*   **New**: **"Architecting the Future of Mobile. Secure. Scalable. Intelligent."**
*   **Sub-headline**: "Senior Mobile Architect specializing in high-performance ecosystems, advanced security hardening, and AI-first engineering."

## 3. The "Deep Dive" Project Cards
Instead of just a description, each card should have a "Technical Specs" drawer:
*   **Architecture**: "BLoC / Clean Architecture"
*   **Secret Sauce**: "Custom Frida-resistant root detection"
*   **Performance**: "99.9% crash-free sessions across 100k users"
*   **AI Stack**: "LLM orchestration via custom MCP server"

## 4. The "Authority Section" (New)
Add a section titled **"Engineering Philosophy"** or **"The Captain's Code"**:
*   **Security First**: "If it's not hardened, it's not finished."
*   **User Centric**: "Architecture is invisible; UX is undeniable."
*   **Constant Voyage**: "The ocean of tech is infinite; curiosity is the only compass."

## 5. Blog Integration Enhancements
*   Add a **"Reading Time"** and **"Difficulty Level"** (e.g., Intermediate, Architect-Level) to each blog post.
*   Feature your **MCP / AI Memory** blog as a "Must Read" to signal your edge in 2026.

## 6. Interactive Element: "The Console"
Add a small interactive terminal or console at the bottom where visitors can "run commands" to see your hidden stats (e.g., `git stats`, `fav_anime`, `current_mission`). This reinforces the "hacker" identity.
